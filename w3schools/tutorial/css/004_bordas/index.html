CSS Border Style

A propriedade border-style especifica que tipo de borda exibir.

Os seguintes valores são permitidos:

  • dotted - Define uma borda pontilhada
  • dashed - Define uma borda tracejada
  • solid - Define uma borda sólida
  • double - Define uma borda dupla
  • groove - Define uma borda ranhurada 3D. O efeito depende do valor da cor da borda
  • ridge - Define uma borda ondulada 3D. O efeito depende do valor da cor da borda
  • inset - Define uma borda de inserção 3D. O efeito depende do valor da cor da borda
  • outset - Define uma borda inicial 3D. O efeito depende do valor da cor da borda
  • none - Não define borda
  • hidden - Define uma borda oculta

A propriedade border-style pode ter de um a quatro valores(para a borda superior, borda direita, borda inferior e borda esquerda).

CSS Border Width

A propriedade border-width especifica a largura das quatro bordas.

A largura pode ser definida como um tamanho específico (px, pt, cm, em, etc) ou usando um dos três valores predefinidos: thin(fina), medium(média) ou thick(grossa):

Exemplos:

thin

medium

thick

2px 4px 6px 8px (top, right, bottom, right)

2px 4px (2px top/bottom e 4px right/left)

CSS Border Color

A propriedade border-color é usada para definir a cor das quatro bordas.

Exemplo:

    
    p.one {
        border-style: solid;
        border-color: red;
      }
      
      p.two {
        border-style: solid;
        border-color: green;
      }
      
      p.three {
        border-style: dotted;
        border-color: blue;
      }

Resultado:

A propriedade border-color pode ter de um a quatro valores(top, right, bottom, left)

red blue (top/bottom e right/left)

CSS Shorthand Border Property

Como você viu até aqui, há muitas propriedades a serem consideradas ao lidar com bordas.

Para encurtar o código, também é possível especificar todas as propriedades de borda individuais em uma propriedade.

A propriedade border é uma propriedade abreviada para as seguintes propriedades de borda individuais:

  1. border-width
  2. border-style(obrigatório)
  3. border-color

Exemplo

    
    p {
        border: 2px solid grey;
        }
