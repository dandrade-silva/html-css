CSS Margins

As margens são usadas para criar espaço ao redor dos elementos, fora de qualquer borda definida.

Este elemento tem uma margem de 70px.

Com CSS, você tem controle total sobre as margens. Existem propriedades para definir a margem para cada lado de um elemento (superior, direita, inferior e esquerda).

Todas as propriedades de margem podem ter os seguintes valores:

  • auto - o navegador calcula a margem
  • length - especifica uma margem em px, pt, cm, etc.
  • % - especifica uma margem em % da largura do elemento recipiente
  • inherit - especifica que a margem deve ser herdada do elemento pai

Dica: Valores negativos são permitidos.

CSS Margin Collapse

Às vezes, as margens superior e inferior dos elementos são recolhidas em uma única margem que é igual à maior das duas margens.

Isso não acontece nas margens esquerda e direita! Apenas as margens superior e inferior!

Observe o seguinte exemplo:

Exemplo

Demonstração de colapso de margem:

        
h1 {
    margin 0 0 50px 0;
}
    
h2 {
    margin 20px 0 0 0;
}
