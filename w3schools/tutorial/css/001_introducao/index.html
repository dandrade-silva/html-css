Primeiros passos CSS

Sintaxe CSS

h1{color: blue; font-size: 12px;}

Explicação:

  1. *{ = Início do bloco de declaração
  2. h1 = O seletor aponta para o elemento HTML que você deseja estilizar.
  3. **color:blue; = declaração com "propriedade:valor" separadas por ponto e vírgula (;).
  4. **font-size:12px = declaração com "propriedade:valor" separadas por ponto e vírgula (;).
  5. *} = Final do bloco de declaração

*Várias declarações CSS são separadas por ponto e vírgula e os blocos de declaração são cercados por chaves.

**Cada declaração inclui um nome de propriedade CSS e um valor, separados por dois pontos.

Como adicionar CSS ao HTML

  • CSS Externo
    • A página HTML deve incluir uma referência ao arquivo CSS (<link rel="stylesheet" href="nomearquivo.css">) dentro da seção head.
  • CSS Interno
    • O estilo interno é definido dentro do elemento <style>, dentro da seção head.
  • CSS Embutido
    • Adicione o atributo style ao elemento relevante

Códigos de Cores

  • RGB (red, green, blue)

    Cada parâmetro define a intensidade da cor entre 0 e 255.

    Os tons de cinza geralmente são definidos usando valores iguais para todas as 3 fontes de luz:

    Cor rgb(255,0,30)

    Cor rgb(70,60,00)

  • RGBA (red, green, blue, alfa)

    Os valores de cores RGBA são uma extensão dos valores de cores RGB com um canal alfa - que especifica a opacidade de uma cor. O parâmetro alfa é um número entre 0,0 (totalmente transparente) e 1,0 (nada transparente):

    Cor rgba(255,60,70,1)

    Cor rgba(80,80,80,0.2)

  • HEX (#RRGGBB)

    Uma cor hexadecimal é especificada com: #RRGGBB, onde os inteiros hexadecimais RR (red), GG (green) e BB (blue) especificam os componentes da cor.

    Exemplo com a cor #ff0000

    Exemplo com a cor #fa05a0

  • HSL/HSLA (hue, saturation, lightness, alfa)

    Em CSS, uma cor pode ser especificada usando matiz, saturação e luminosidade (HSL)

    Matiz é um grau na roda de cores de 0 a 360. 0 é vermelho, 120 é verde e 240 é azul.

    A saturação é um valor percentual. 0% significa um tom de cinza e 100% é a cor total.

    A luminosidade também é uma porcentagem. 0% é preto, 50% não é nem claro nem escuro, 100% é branco
